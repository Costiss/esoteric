Esotheric mobile UI to match the mystical, high-fidelity aesthetic of apps like Stardust.
🎨 Design & Technical Plan

1.  Refined "Deep Void" Theme:
    - Background: Deepest midnight purple (#050208) with subtle radial gradients.
    - Accents: "Nebula Red" (#FF2D55), "Electric Cyan" (#00F2FF), and "Ethereal Green" (#39FF14).
    - Typography: Elegant, high-contrast serif for headings; clean sans-serif for utility.
2.  Mystical Animation Suite:
    - Stardust Background: A persistent, low-CPU canvas component with twinkling particles.
    - Haptic & Visual Feedback: Soft glows and scale-springs on press (using Reanimated).
    - Magic Transitions: Shared element-like transitions and layout animations for screen entries.
    - Sparkle Overlays: Subtle "sparkle" generators for successful actions (e.g., booking confirmation).
3.  Modern Layout Overhaul:
    - Glassmorphism (frosted glass) cards for service listings.
    - Floating navigation with blurred backgrounds.
    - Interactive "tarot-card" style layouts for services and providers.
